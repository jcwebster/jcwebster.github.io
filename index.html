In this project, I combined my interest in image processing and my love of algorithm to create a program that takes and image of a maze and finds the optimal solution. Gesture Controlled Robotic Manipulator

This project utilized a Myo Armband to control a remote robotic manipulator via Bluetooth. By using onboard accelerometers, we captured the movement of the users arm. This movement was translated via a C++ program to commands which were sent over Bluetooth to the robot. Using this control scheme, the user was able to drive, raise and lower the arm and rotate the claw using the motion of their arm. They were also able to grip and release a beaker using hand gestures, which were picked up by the EMG sensors on the myo.
